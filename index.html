This application was developed as my dissertation project for University. I created an augmented reality application for Lincoln Cathedral using Googles ARCore and Unity. Many sections were created through combining Unitys physics engine capabilities with ARCore elements to augment objects and allow the user to iteract with them.

My Food Tracker

I developed this app for a cross-platform development module at university. We used react-native combined with DOM elements to produce the application Each application needed to be related to one specific area in particular and mine was sustainability. I created an application that allowed the user to store information about items they had bought that would expire. The app uses various API's to access the device calendar, notification system and camera to allow users to store these items and send reminders that will let them know when their items expire.
